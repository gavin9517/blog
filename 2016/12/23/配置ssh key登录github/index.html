配置ssh key登录github

创建SSH Key

  1. 打开终端

  2. 输入以下命令 

    $ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

    根据输入的邮箱创建一个ssh key,填入邮箱之后会出现以下的提示信息。这里我们需要设置一个自定义名词。例如：id_rsa_work

    $ Enter a file in which to save the key (/Users/you/.ssh/id_rsa): [Press enter]

    执行完毕之后会生成 id_rsa_work 和 id_rsa_work.pub 两个文件。

添加生成的SSH Key到ssh-agent中

执行以下命令添加ssh key到到 ssh-agent 中。

$ ssh-add ~/.ssh/id_rsa_work

这里，id_rsa_work就是上面自定义的证书名词。

配置ssh

打开 ~/.ssh/config 文件(如果没有则创建一个)。编辑配置文件，使得能够自动识别该用哪个证书。注意缩进

Host work
    HostName github.com
    User git
    IdentityFile ~/.ssh/id_rsa

Host github.com
    HostName github.com
    User git
    IdentityFile ~/.ssh/id_rsa_person

部署到 github 上

  1. 点击顶部的 Setting 链接。
  2. 在账户左边的侧边,点击 SSH and GPG keys
  3. 点击 New SSH key.这里别点错了。
  4. 给 SSH KEY 取个名字.然后,复制文件 id_rsa_work.pub 的内容到输入框，点击add ssh key保存就好了。

添加源

最后添加源就可以了。

git remote add origin git@github.com:gavin9517/gavin9517.github.com.git
