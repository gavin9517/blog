css中font smoothing属性用法

今天访问larvel官网的时候觉得字体看起来很舒服。查看使用字体时发现了-webkit-font-smoothing这个属性。查询文档得知这个属性可以让MacOS字体更好看(平滑)。
注意的是该属性只能在Mac OS X 平台下生效。

-webkit-font-smoothing主要用到有两个属性subpixel-antialiased和antialiased。
subpixel-antialiased是次级像素平滑。
antialiased是灰度平滑。
只要不是none在MacOS下看到都蛮舒服的。对于非正式项目可以用来试试。
